Arquivo de Referência HTML e CSS de Magno

HTML

Tags Básicas

  • <!DOCTYPE html>: Define o tipo de documento como HTML5.
  • <html>: Define o início e o fim do documento HTML.
  • <head>: Contém informações sobre o documento HTML.
  • <title>: Define o título do documento, que aparece na aba do navegador.
  • <body>: Contém o conteúdo do documento visível para o usuário.
  • <h1> a <h6>: Define cabeçalhos de nível 1 a 6.
  • <p>: Define um parágrafo de texto.
  • <a>: Define um link para outra página ou recurso.
  • <img>: Define uma imagem a ser exibida na página.
  • <ul> e <li>: Define uma lista não ordenada.
  • <ol> e <li>: Define uma lista ordenada.
  • <br>: Insere uma quebra de linha.
  • <hr>: Insere uma linha horizontal.

Atributos Básicos

  • class: Define uma classe CSS para um elemento.
  • id: Define um identificador exclusivo para um elemento.
  • style: Define o estilo CSS inline para um elemento.
  • href: Define o link para um elemento <a>.
  • src: Define o caminho para uma imagem em um elemento <img>.
  • alt: Define um texto alternativo para uma imagem em um elemento <img>.

CSS

Seletores

  • element: Seleciona todos os elementos de um tipo.
  • #id: Seleciona o elemento com o ID especificado.
  • .class: Seleciona todos os elementos com a classe especificada.
  • element, element: Seleciona vários elementos.
  • element.class: Seleciona todos os elementos de um tipo com uma classe especificada.
  • element > element: Seleciona um elemento filho de outro.
  • element + element: Seleciona o elemento imediatamente após outro.
  • element ~ element: Seleciona todos os elementos irmãos após o primeiro elemento especificado.
  • :hover: Seleciona um elemento quando o mouse está sobre ele.
  • :active: Seleciona um elemento quando ele é clicado.
  • :visited: Seleciona um link visitado.

Propriedades

  • color: Define a cor do texto.
  • font-size: Define o tamanho da fonte.
  • font-weight: Define o peso da fonte.
  • text-align: Define o alinhamento do texto.
  • background-color: Define a cor de fundo.
  • border: Define a borda.
  • padding: Define o espaçamento interno.
  • margin: Define o espaçamento externo.
  • display: Define como o elemento é exibido.
  • float: Define o posicionamento flutuante do elemento.
  • clear: Define o comportamento de limpeza dos elementos flutuantes.

Exemplo de Uso de CSS

Este é um exemplo de como usar CSS em um arquivo HTML:

Este exemplo usa uma fonte sans-serif

<html>
<head>
	<style>
		h1 {
			color: blue;
			text-align: center;
		}
		p {
			font-size: 18px;
		}
		.container {
			width: 80%;
			margin: 0 auto;
		}
	</style>
</head>
<body>
	<div class="container">
		<h1>Título</h1>
		<p>Este é um parágrafo.</p>
	</div>
</body>
</html>
